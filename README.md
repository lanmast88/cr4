#Отчёт по контрольной работе

В рамках контрольной работы я разработал небольшое React-приложение. Задача заключалась в том, чтобы показать понимание компонентов, работы с состоянием, обработкой событий, а также использование локального хранилища и базовой архитектуры приложения.

##Описание приложения

Моё приложение представляет собой простой сервис для управления рецептами. Пользователь может добавлять новые рецепты, указывать описание, просматривать список уже созданных рецептов.

##Основные возможности

добавление нового рецепта через форму;

отображение всех рецептов в виде списка;

##Особенности реализации

Приложение написано на React с использованием функциональных компонентов и хуков. Для состояния я использовал useState, а для сохранения данных — useEffect и localStorage. Также был создан небольшой компонент для формы и отдельный компонент для списка рецептов.

##Вывод

В ходе работы я закрепил использование основных возможностей React, разобрался с работой состояния, обработкой пользовательских действий.
